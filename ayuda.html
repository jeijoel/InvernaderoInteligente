Ayuda y Soporte

Preguntas Frecuentes

¿Cómo ingreso al sistema?

Ingrese su usuario o ID de dispositivo y su contraseña, luego haga clic en Entrar.

¿Qué hago si olvidé mi contraseña?

Actualmente no hay recuperación automática. Comuníquese con el administrador del sistema.

¿Cómo me registro?

Presione el botón Registrarse y complete el formulario con sus datos.

¿Qué pasa si tengo problemas técnicos?

Revise su conexión a Internet. Si el problema persiste, contacte al equipo de soporte técnico a través de la sección Tickets Soporte de la aplicación.

Manual de Usuario

Panel de Sensores

Esta sección muestra en tiempo real los datos recopilados por los sensores del invernadero:
- Temperatura: Muestra la temperatura ambiente actual en grados Celsius.
- Humedad Suelo: Indica si el suelo está "Húmedo" o "Seco".
- Nivel Agua: Muestra el estado del tanque de agua ("Lleno", "Medio", "Vacío").
- Iluminación: Indica el nivel de luz ("Luz", "Oscuro").

Presione Ver Gráficas para visualizar el historial de datos de cada sensor. Use el botón Actualizar Datos de Sensores para refrescar la información manualmente.

Alertas Automáticas

La aplicación le notificará con una ventana emergente si detecta condiciones críticas, como temperaturas muy altas (>35°C) o bajas (<15°C), suelo seco, nivel de agua bajo o falta de iluminación.

Control Manual de Actuadores

Permite controlar directamente los componentes del invernadero:
- Techo: Use el botón Abrir/Cerrar Techo para controlar la cubierta superior. El estado actual (Abierto/Cerrado) se muestra al lado.
- Luz: Use Encender Luz / Apagar Luz para controlar la iluminación artificial.
- Ventilador: Use Encender / Apagar para activar o desactivar el ventilador. El estado se muestra al lado.

Configuración de Ciclos Automáticos

Aquí puede configurar el comportamiento automático del sistema. Ajuste los deslizadores para definir los siguientes intervalos en segundos:
- Intervalo Datos: Define cada cuántos segundos el sistema central registrará los datos de los sensores.
- Intervalo Luz: Define la duración que la luz permanecerá encendida y luego apagada en un ciclo.
- Intervalo Ventilador: Define la duración que el ventilador permanecerá encendido y luego apagado en un ciclo.

Después de ajustar los intervalos, es muy importante presionar Guardar Intervalos para enviar la nueva configuración al dispositivo. Use Iniciar Ciclos para comenzar la operación automática y Detener Ciclos para pararla en cualquier momento.

Funciones Adicionales

- Ver Cámara: Abre una ventana para visualizar la transmisión de la cámara web instalada en el invernadero.

- Tickets Soporte: Abre la interfaz para crear y gestionar tickets de soporte técnico con el administrador.

- Cerrar Aplicación: Es importante usar este botón para cerrar el programa. Así se asegura que la conexión con el hardware se termine correctamente y se detengan todos los procesos en segundo plano.
